# Xadrez - pygame

O objetivo desse projeto é fazer um jogo de xadrez onde é possivel jogar contra a engine [Stockfish](https://stockfishchess.org/)

## Requisitos
```
$ pip install -r requirements.txt
```
- Informe o caminho do executavel do stockfish
